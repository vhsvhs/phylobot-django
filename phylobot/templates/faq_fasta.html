Notes on FASTA Files

PhyloBot reads input sequences in the FASTA file format. This format supports sequence collections that are not aligned. Every line in the FASTA file should be either a sequence name or sequence content. Sequence name lines begin with the character '>', followed by the name of the sequence. Sequence content lines contain only amino acid or nucleotide strings. Sequence content lines may contain insertion/deletion characters (i.e. '-'), but PhyloBot will trim out these indels before running the alignment algorithms. If a sequence content is very long, you can wrap the seqeunce into multiple contiguous lines.
